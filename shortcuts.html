VSCode Keyboard Shortcuts

Shortcut (Windows/Linux) Shortcut (Mac) Explanation Example
Ctrl + P Cmd + P Quickly open files by name. Type part of a filename to open it.
Ctrl + Shift + P Cmd + Shift + P Open Command Palette to access all commands. Type "format" to find formatting commands.
Ctrl + / Cmd + / Toggle line comment. Comment or uncomment the current line.
Alt + Up/Down Option + Up/Down Move line up or down. Reorder lines of code quickly.
Shift + Alt + Down Shift + Option + Down Copy line down. Duplicate the current line below.
Ctrl + Shift + K Cmd + Shift + K Delete line. Remove the current line.
Ctrl + Space Cmd + Space Trigger suggestion/intellisense. Show code completion suggestions.
Ctrl + Shift + O Cmd + Shift + O Go to symbol in file. Jump to a function or class in the file.
Ctrl + B Cmd + B Toggle sidebar visibility. Show or hide the explorer/sidebar.
Ctrl + ` Cmd + ` Toggle integrated terminal. Open or close the terminal panel.
Ctrl + Shift + F Cmd + Shift + F Search in all files. Find text across the workspace.
Ctrl + F Cmd + F Find in file. Search for text in the current file.
Ctrl + H Cmd + Option + F Replace in file. Find and replace text in the current file.
F2 F2 Rename symbol. Change variable/function/class name everywhere.
Ctrl + D Cmd + D Select next occurrence of word. Multi-cursor editing for repeated words.
Ctrl + Shift + L Cmd + Shift + L Select all occurrences of word. Edit all instances of a word at once.
Ctrl + Shift + M Cmd + Shift + M Show problems panel. View errors and warnings in your code.
Ctrl + Tab Cmd + Option + Right Switch to next open editor tab. Cycle through open files.
Ctrl + Shift + N Cmd + Shift + N New window. Open a new VSCode window.
Ctrl + W Cmd + W Close editor tab. Close the current file.

For a full list, see official VSCode keyboard shortcuts.
